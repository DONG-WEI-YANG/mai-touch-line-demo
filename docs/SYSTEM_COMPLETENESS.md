## 🚀 NLP 服務架構 (新增)

### 系統架構
```
┌─────────────────────────────────────────┐
│         Main Application (Node.js)       │
│         Express + tRPC                   │
└─────────────┬───────────────────────────┘
              │ HTTP REST API
              ▼
┌─────────────────────────────────────────┐
│      NLP Service (Python FastAPI)       │
├─────────────────────────────────────────┤
│  ┌─────────────────────────────────┐   │
│  │      Model Pool Manager          │   │
│  │  - Load Balancing                │   │
│  │  - Health Checking               │   │
│  │  - Auto Recovery                 │   │
│  └─────────────────────────────────┘   │
│                                          │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐│
│  │ Model 1  │ │ Model 2  │ │ Model 3  ││
│  │ (Intent) │ │(Sentiment)│ │ (Entity) ││
│  └──────────┘ └──────────┘ └──────────┘│
└─────────────────────────────────────────┘
```

### 特性
- **獨立虛擬環境**: 不影響全域 Python 環境
- **多模型池化**: 3個並行實例，自動負載均衡
- **輕量級模型**: 基於 DistilBERT (~250MB)
- **高性能**: 平均延遲 50-100ms
- **自動恢復**: 錯誤實例自動重啟
- **健康監控**: 實時統計和監控
